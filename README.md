# SecureForge 🔐
A Python password generator and strength checker.

## Features
- Generate secure passwords with custom length and character types
- Check strength of any password with a score out of 8
- Suggests improvements for weak passwords

## How to run
```bash
python secureforge.py
```

## Requirements
- Python 3.x
- No external libraries needed (uses built-in `string` and `random`)
